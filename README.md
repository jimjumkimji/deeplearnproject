ตรวจจับข้อความเชิงลบด้วย ML
		
ยุคนี้เป็นยุคของเทคโนโลยีและโซเชียลมีเดียทุกคนสามารถติดต่อกันได้ทั่วโลกไม่ว่าจะอยู่ไหนบนโลก สามารถพูดคุยแลกเปลี่ยนความคิดเห็นโดยไม่จำกัดเขตแดน ซึ่งก็เป็นข้อดีอย่างมากในหลายๆด้าน แต่ในอีกมุมนึงก็มีผู้หวังไม่ดี ที่ตั้งใจโพสต์ หรือแสดงความเห็นที่ไม่ได้มีเหตุและผล ตั้งใจเพียงแค่กลั่นแกล้งผู้อื่น เนื่องด้วยผมเห็นเหตุการณ์แบบนี้มาหลายครั้งที่ชอบมีคนโพสต์ หรือแสดงความเห็นในโลกออน์ไลน์ในเชิงลบที่จงใจจะกลั่นแกล้งผู้อื่น ประกอบกับได้มีโอกาสทำ project deeplearn จึงสนใจจะทำ model ที่จะตรวจจับข้อความเชิงลบในโลกออนไลน์

จุดเด่น
ส่วนมากในโซเชี่ยลมีเดียจะมีระบบตรวจจับคำหยาบคาย แต่บางที่ถ้อยคำเชิงลบหรือเชิงกลั่นแกล้งไม่ได้ออกมาแค่ในรูปแบบคำหยาบคายตรงๆ ทำให้ระบบไม่มารถจับได้ ตัวโมเดลนี้ก็จะเข้ามาเพื่อแก้ปัญหาตรงนี้ได้ 
จุดด้อย
อาจจะไม่แม่นยำเท่ามนุษย์เพราะ โลกก็มีการเปลี่ยนแปลงไปทุกวัน มีคำศัพท์ใหม่ๆทุกวัน ประกอบกับความซับซ้อนของความหมายแต่ละคำของมนุษย์ ทำให้บางคำที่ซับซ้อน ซ่อนความหมาย ตัว model นี้ก็อาจจะยังไม่สามารถจับได้ครับ
